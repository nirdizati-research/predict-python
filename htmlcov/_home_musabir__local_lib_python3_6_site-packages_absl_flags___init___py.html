15"""This package is used to define and parse command line flags. 

16 

17This package defines a *distributed* flag-definition policy: rather than 

18an application having to define all flags in or near main(), each Python 

19module defines flags that are useful to it. When one Python module 

20imports another, it gains access to the other's flags. (This is 

21implemented by having all modules share a common, global registry object 

22containing all the flag information.) 

23 

24Flags are defined through the use of one of the DEFINE_xxx functions. 

25The specific function used determines how the flag is parsed, checked, 

26and optionally type-converted, when it's seen on the command line. 

27"""
